CommonLibOgl  - a static library containing commonly used OpenGL functionality, support code for windowing, etc.

Uses libraries:
	- GLEW: http://glew.sourceforge.net/
	- GLM: http://glm.g-truc.net/
	- SOIL: http://www.lonesock.net/soil.html
